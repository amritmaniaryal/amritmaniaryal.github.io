Research Interests

  • Narrative understanding using Large Language Models (LLMs) and their integration with logic programming for structured information extraction.
  • Machine learning applications in remote sensing, including deforestation detection and urban expansion.

Current Projects

  • Deforestation Detection Using Remote Sensing

    Leveraging Landsat satellite imagery to detect deforestation patterns in California using machine learning techniques such as Random Forest and K-Means.

  • Exploiting LLMs with Logic Programming for Narrative Understanding

    Exploring how LLMs can extract structured data from natural language stories and feed it into logic programming systems for reasoning and understanding. This project focuses on leveraging Answer Set Programming (ASP) and prompt engineering techniques to interpret scenarios like restaurant activities or daily routines.
